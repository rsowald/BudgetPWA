![License](https://img.shields.io/badge/license-MIT-green.svg)
  
  ## Description

  [Deployed Application](https://infinite-plateau-82165.herokuapp.com/)
   
   This is a personal budget tracker designed to be used online or offline to allow a user to always know precisely how much they have to spend.

  ## Usage
  1. Navigate to [https://infinite-plateau-82165.herokuapp.com/](https://infinite-plateau-82165.herokuapp.com/) to view the funds tracker.
  1. Record the name or value of any transaction you make, whether it is money earned or spent. (Note: your value should always be a positive integer.)
  1. Click the appropriate button to either subtract or add the value of your transaction.
  1. See your cumulative total change in response.
  1. The chart will also update to show a visual representation of your spending history.
